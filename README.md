# TypeScript-Understanding

perform this commands after clone the repo
and make sure you have installed node, npm.

1. install typescript

- `npm i -g typescript`

2. setup project

- `npm install`

3. start development server

- `npm start`

4. start compilation

- for specific file

- `tsc app.ts` 

- for general project folder:

- `tsc --init` 

- `tsc -w`
